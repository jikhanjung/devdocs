# Noematica Brand Package

**Date:** 2026-04-03  
**Status:** Draft  
**Scope:** Consolidated naming and domain package for the `Noematica` company brand

## 1. Core Brand Decision

The emerging company brand is:

**Noematica**

This name is currently strong because it balances:

- conceptual depth
- scholarly fit
- infrastructure compatibility
- long-term extensibility
- usable Korean pronunciation

It avoids the key limitations identified in other naming directions:

- not too paleontology-specific
- not too old-fashioned like some `Scholar...` names
- not too NLP-coded like `Corpus...`
- not too legal/medical like `Evidence...`

## 2. Core Brand Meaning

`Noematica` is built around the root idea of `noema`:

- meaningful conceptual content
- the content of thought
- a meaning-bearing unit
- a thought-object

Brand-wise, `Noematica` can be understood as a company concerned with:

- meaningful knowledge structures
- scholarly systems
- research corpus infrastructure
- reusable knowledge artifacts

It is broad enough to hold:

- PaperMeister
- SCODA
- PaleoBase
- future scholarly infrastructure products

## 3. Korean Rendering

Recommended Korean rendering:

**노에마티카**

Why it works:

- more natural than many other abstract English candidates
- sounds like a proper brand name
- relatively stable in pronunciation
- neither too rigid nor too playful

## 4. Recommended Domain Structure

### Primary domain

**noematica.ai**

Role:

- main global site
- primary company brand URL
- English-facing home for products and company identity

Strengths:

- direct and easy to understand
- preserves the full brand name
- useful for presenting the company internationally

Risks:

- `.ai` is common and can over-signal “AI company”

Interpretation:

Best used as the practical main domain unless a stronger long-term `.com` option appears later.

### Korean domain

**noematica.kr**

Role:

- Korean-facing site or landing path
- local trust and accessibility
- domestic pilot, partnership, and contact flows

Strengths:

- helps the brand feel grounded in Korea
- useful for local outreach and institutional trust
- supports Korean-language marketing and documentation

Interpretation:

A strong complement to the main domain and worth registering if available.

### Secondary / short domain

**noemati.ca**

Role:

- short branded URL
- redirect domain
- campaign, demo, or business card URL

Strengths:

- visually memorable
- elegant domain hack
- shorter and more stylized than the full brand domain

Risks:

- not everyone will immediately connect it to `Noematica`
- some may initially read `.ca` as a Canada domain rather than a brand extension

Interpretation:

Best used as a supporting brand domain, not the primary home.

## 5. Recommended Domain Roles

The cleanest structure is:

- `noematica.ai` = main company site
- `noematica.kr` = Korean-facing site or redirect path
- `noemati.ca` = short-link / branded redirect

This creates a practical and flexible domain stack.

## 6. Brand Architecture

**Noematica**

### Products / Platforms

- SCODA

## 9. Brand Feel Summary

`Noematica` feels:

- intellectual
- modern
- infrastructure-compatible
- slightly abstract, but not empty
- more like a serious long-term company than a single-product label

It is not the most literal name, but that can be an advantage if the company wants to grow into a broader identity.

## 10. Risks to Watch

Even if the name is favored, several checks still matter.

- domain availability
- trademark conflict screening
- company name search in key jurisdictions
- GitHub / social handle availability
- confusion with similarly sounding names

The brand fit is promising, but final commitment should still be operationally validated.

## 11. Recommended Next Step

If moving forward with `Noematica`, the most useful next actions are:

1. secure the core domains if available
2. test the name in realistic usage:
   - website header
   - company intro sentence
   - email signature
   - business card
   - Korean and English spoken use
3. check naming conflicts before full commitment

The current brand package is coherent:

- **Company:** Noematica
- **Primary domain:** noematica.ai
- **Korean domain:** noematica.kr
- **Short brand domain:** noemati.ca

This combination gives:

- conceptual identity
- practical global usage
- Korean accessibility
- a more memorable short-form brand asset

As a working brand package, it is one of the strongest and most internally consistent naming directions explored so far.
