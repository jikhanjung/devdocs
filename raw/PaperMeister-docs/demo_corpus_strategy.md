# PaperMeister 데모 코퍼스 전략

**날짜:** 2026-04-03  
**상태:** 초안  
**목적:** 제품 유용성과 잠재력을 설득력 있게 보여주기 위한 데모용 corpus 전략 정리

## 1. 문제

PaperMeister의 핵심 가치는 소수 샘플보다, 어느 정도 규모의 corpus가 실제로 읽히고 검색 가능한 상태가 되었을 때 드러난다.

즉, 데모 목적에서도 “몇 편의 PDF OCR”로는 충분하지 않다.

반면 전체 컬렉션을 무차별적으로 처리하는 것은 비용과 법적 리스크 측면에서 비효율적일 수 있다.

따라서 데모용 corpus는 별도로 전략적으로 설계할 필요가 있다.

## 2. 데모 코퍼스의 역할

좋은 데모 코퍼스는 단순 테스트셋이 아니다.

그것은 다음을 동시에 보여주는 showcase asset이다.

- OCR 품질
- 검색 가능성
- 구조 보존 가치
- corpus로서의 응집성
- 향후 분석/추출 확장 가능성

## 3. 권장 방향

초기 데모 코퍼스는 다음 특성을 가지는 것이 좋다.

- 한 분야 또는 가까운 하위 분야 중심
- 500~1000편 수준의 적당한 규모
- 텍스트 PDF와 스캔 PDF가 혼합됨
- figure/caption/section 구조가 중요한 문헌 포함
- 실제 연구자에게도 쓸모 있어 보이는 범위

즉, “넓고 얇은 범용 corpus”보다  
**좁고 깊은 showcase corpus**가 더 설득력이 크다.

## 4. 왜 분야별 코퍼스가 좋은가

분야별 corpus는 다음 장점이 있다.

- 검색 질의가 더 자연스럽다
- 키워드와 용어 분포가 응집적이다
- 향후 엔티티/관계 추출 가능성을 보여주기 쉽다
- 도메인별 pain point를 구체적으로 설명할 수 있다

예:

- 고생물학: taxon, locality, stratigraphy, caption 가치가 큼
- 생물학: morphology, species, methods, supplementary relevance
- 스캔 문헌 중심 분야: OCR salvage 가치가 더 큼

## 5. 내부용과 외부용 구분

데모 코퍼스는 두 종류로 나눠 생각하는 것이 좋다.

## 5.1 내부용 코퍼스

목적:

- OCR 품질 검증
- 구조화 실험
- 검색 성능 점검
- 비용 측정

## 5.2 외부용 코퍼스

목적:

- 제품 가치 설명
- before/after 시연
- 분야별 활용 예시 제공
- 파일럿 고객 설득

외부용 코퍼스는 내부용보다 더 curated되어야 한다.

## 6. 데모에서 보여줄 것

외부 데모에서는 단순히 PDF를 여는 화면보다 다음이 더 중요하다.

- 검색 결과 품질
- 특정 용어에 대한 hit distribution
- 자주 등장하는 키워드와 용어
- 구조화된 문서 예시
- caption/body/section 구분 사례
- 향후 분석 가능성을 보여주는 리포트형 결과

즉, 데모의 핵심은 “OCR이 잘 됐다”가 아니라  
**논문들이 살아 있는 corpus처럼 작동한다**는 점이다.

## 7. 권장 실행 계획

1. 대표 분야 1개 선택
2. 500~1000편 수준의 curated corpus 구성
3. Chandra2 기반 고품질 OCR 수행
4. raw JSON + searchable DB + 샘플 질의 세트 준비
5. before/after 시연 시나리오 구성
6. 이후 다른 분야 코퍼스로 확장

## 8. 요약

PaperMeister의 데모 전략은 대량 범용 처리보다:

- 분야별
- curated
- 구조 가치가 드러나는
- 검색과 분석 가능성을 동시에 보여주는 corpus

를 먼저 만드는 것이 좋다.

즉, 초기 데모 코퍼스는 제품 설명 자료가 아니라  
**제품 가치 자체를 체감하게 만드는 핵심 자산**이다.
