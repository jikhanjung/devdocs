## 1. 시작하기

### 1.1 로그인

1. https://fsis.psok.or.kr 접속
2. 상단 네비게이션 바 우측의 로그인 폼에 아이디/비밀번호 입력
3. [로그인] 버튼 클릭

> 계정이 없는 경우 관리자에게 계정 생성을 요청하세요.

### 1.2 화면 구성

로그인 후 상단 네비게이션 바에 다음 메뉴가 표시됩니다:

| 메뉴 | 설명 |
|------|------|
| **KOFHIN** | 메인 페이지 (FSIS 표본 목록) |
| **화석표본** | 표본목록, 표본지도, 변경내역 |
| **화석산지** | 산지목록, 산지지도, 위험도 대시보드, 모식표본 목록 |
| **KPRDB** | 논문목록, 저자목록, 저널목록 |
| **참조** | 지질시대표, 지층정보, 학명정보 |
| **사용자명** | 내 정보, 로그아웃 |

### 1.3 작업 흐름 개요

논문 데이터 입력은 다음 순서로 진행합니다:

```
1. 저자 등록 (미등록 저자인 경우)
2. 저널 등록 (미등록 저널인 경우)
3. 논문 등록 (PDF 업로드 포함)
4. 분류군(Taxa) 입력
5. 표본 정보 입력 (표본번호, 위경도 등)
```

## 2. 논문 등록

### 2.1 논문 목록 확인

1. **KPRDB** > **논문목록** 클릭
2. 검색창에 저자명, 제목, 연도 등으로 검색하여 이미 등록된 논문인지 확인
3. 등록되지 않은 논문이면 다음 단계로 진행

### 2.2 저자 등록 (필요 시)

논문 등록 전에 저자가 시스템에 등록되어 있어야 합니다.

1. **KPRDB** > **저자목록** 클릭
2. 검색창에서 저자명 검색
3. 미등록 저자인 경우 **[New]** 버튼 클릭
4. 입력 항목:

| 필드 | 설명 | 예시 |
|------|------|------|
| 이름(영문) | 성, 이름 순 | `Park, Tae-Yoon` |
| 이름(한글) | 한글 이름 (선택) | `박태윤` |
| 소속 | 소속 기관 (선택) | `한국지질자원연구원` |

5. [Save] 클릭

### 2.3 저널 등록 (필요 시)

1. **KPRDB** > **저널목록** 클릭
2. 검색 후 미등록이면 **[New]** 버튼 클릭
3. 입력 항목:

| 필드 | 설명 | 예시 |
|------|------|------|
| 저널명(영문) | 정식 영문 저널명 | `Journal of Paleontology` |
| 저널명(한글) | 한글명 (선택) | |
| 약칭 | 저널 약칭 | `J. Paleontol.` |
| ISSN | ISSN 번호 (선택) | `0022-3360` |

4. [Save] 클릭

### 2.4 논문 등록

1. **KPRDB** > **논문목록**에서 **[New]** 버튼 클릭
2. 입력 항목:

#### 기본 정보

| 필드 | 필수 | 설명 | 예시 |
|------|:----:|------|------|
| 영문제목 | O | 논문 영문 제목 | `Late middle Cambrian trilobite faunas...` |
| 한글제목 | | 한글 제목 (한글 논문인 경우) | `조선누층군 세송층 하부의...` |
| 발표년도 | O | 4자리 연도 | `2013` |
| 권 | | Volume 번호 | `87` |
| 호 | | Issue 번호 | `6` |
| 페이지 | | 시작-끝 페이지 | `991-1003` |
| 저널 | O | 드롭다운에서 선택 (자동완성) | `Journal of Paleontology` |
| 언어 | | English / Korean | `EN` |
| DOI | | DOI 번호 (있는 경우) | `10.1017/jpa.2013.64` |

#### 분류 정보

| 필드 | 설명 | 선택값 |
|------|------|--------|
| 문헌종류 | 문헌 유형 | Journal Article / Book / Report / Thesis |
| 분류유형 | 논문 성격 | 분류학(표본번호 있음) / 분류학(표본번호 없음) / 비분류학 / 해외화석 |
| 화석종류 | 화석 분류 | 거화석 / 미화석 / 생흔화석 |
| 일반명 | 대표 화석군 | 삼엽충, 이매패류, 복족류, 공룡 등 (복수 선택 가능) |

#### 저자 연결

- 논문 등록 폼 하단에 저자 추가 영역이 있습니다
- 자동완성 필드에 저자명 입력 → 목록에서 선택
- **저자 순서**를 정확히 입력 (1저자, 2저자, ...)
- 여러 저자를 추가하려면 [+] 버튼 클릭

#### PDF 업로드

- **파일** 필드에서 논문 PDF 파일 선택
- PDF가 있으면 시스템이 자동으로 텍스트 추출, 좌표/표본번호 추출을 시도합니다

3. 모든 항목 입력 후 **[Save]** 클릭

### 2.5 논문 수정

1. **논문목록**에서 논문 제목 클릭 → 상세 페이지
2. **[Edit]** 버튼 클릭
3. 수정 후 **[Save]** 클릭

## 3. 분류군(Taxa) 입력

논문에 기재된 분류군(종, 속 등)을 입력합니다.

### 3.1 분류군 추가

1. **논문 상세 페이지** 하단의 분류군 테이블에서 **[Edit Taxa]** 클릭
2. 분류군 편집 폼이 열립니다
3. 각 분류군에 대해 입력:

| 필드 | 설명 | 예시 |
|------|------|------|
| Taxon 이름 | 학명 (자동완성) 또는 직접 입력 | `Redlichia chinensis` |
| 층 | 지층명 (자동완성) | `Sesong Formation` |
| 시대 | 지질시대 (자동완성) | `Cambrian Series 3` |
| 표본사진 수 | 피겨가 있는 표본 수 | `5` |
| 기타 표본 수 | 피겨 없는 표본 수 | `12` |
| 비고 | 추가 메모 | |

4. 여러 분류군을 추가하려면 행 추가
5. **[Save]** 클릭

### 3.2 학명 등록 (필요 시)

자동완성에서 학명이 검색되지 않으면:

1. **참조** > **학명정보** > **[New]** 클릭
2. 입력 항목:

| 필드 | 설명 | 예시 |
|------|------|------|
| 학명 | 속명 + 종명 | `Redlichia chinensis` |
| 저자 | 명명 저자 | `Walcott, 1905` |

3. [Save] 후 다시 분류군 편집으로 돌아가 자동완성에서 선택

## 4. 표본 정보 입력

각 분류군에 속하는 개별 표본의 상세 정보를 입력합니다.

### 4.1 표본 정보 편집 화면 접근

1. **논문 상세 페이지** 하단의 분류군 테이블에서 원하는 분류군의 **연필 아이콘(편집)** 클릭
2. 표본 편집 폼이 열립니다

### 4.2 표본 정보 입력

각 표본에 대해 다음을 입력합니다:

| 필드 | 설명 | 예시 |
|------|------|------|
| 수장기관표본번호 | 기관코드-번호 형식 | `KIGAM-F-2024-001` |
| 피겨번호 | 논문 내 피겨 번호 | `Fig.3-A` |
| 위치(행정구역) | 산출지 행정구역 | `경상남도 진주시 정촌면` |
| 위도 | DMS 또는 decimal | `35°07'45''N` |
| 경도 | DMS 또는 decimal | `128°06'02''E` |
| 비고 | 추가 메모 | `Holotype` |

### 4.3 위경도 입력 규칙

위경도는 다음 형식 중 하나로 입력합니다:

| 형식 | 위도 예시 | 경도 예시 |
|------|-----------|-----------|
| DMS (도분초) | `35°07'45''N` | `128°06'02''E` |
| DMS (소수점) | `35°07'45.5''N` | `128°06'02.3''E` |
| Decimal | `35.129167` | `128.100556` |

> **중요**: 위경도는 화석산지 데이터와 연결되는 핵심 정보입니다. 논문에 기재된 좌표를 정확히 입력해주세요.

### 4.4 표본 목록 확인

- 분류군 행의 **확장 아이콘(↕)** 클릭 → 해당 분류군의 표본 목록이 펼쳐집니다
- 표본번호, 피겨번호, 위치, 위경도, 비고를 확인할 수 있습니다

## 5. 보조 데이터 관리

### 5.1 지층 정보

1. **참조** > **지층정보** 클릭
2. 분류군 입력 시 자동완성에서 지층이 없으면 **[New]** 클릭하여 추가

| 필드 | 설명 | 예시 |
|------|------|------|
| 이름(영문) | 지층 영문명 | `Jigunsan Formation` |
| 이름(한글) | 지층 한글명 | `직운산층` |
| 단위 | Formation / Group / Member 등 | `Formation` |

### 5.2 지질시대

- **참조** > **지질시대표**에서 계통도를 확인할 수 있습니다
- 지질시대 데이터는 사전 등록되어 있으며, 일반사용자는 추가할 필요가 거의 없습니다
- 필요 시 관리자에게 요청하세요

## 6. 입력 가이드라인

### 6.1 논문 분류유형 선택 기준

| 유형 | 설명 | 예 |
|------|------|-----|
| **분류학(표본번호 있음)** | 표본 기재 논문, 표본번호가 명시됨 | 신종 기재, 재기재 |
| **분류학(표본번호 없음)** | 분류학적 내용이나 표본번호 미명시 | 화석군 개관, 생층서 |
| **비분류학** | 화석 관련이나 분류학이 아닌 논문 | 퇴적환경, 고환경 복원 |
| **해외화석** | 한반도 외 지역의 화석 | 비교 목적 등록 |

### 6.2 표본번호 작성 규칙

- 논문에 기재된 표본번호를 **그대로** 입력합니다
- 기관 약칭이 포함된 경우 그대로 유지: `KIGAM-F-001`, `SNU-01234`
- 동일 표본이 여러 논문에 등장하면 **각 논문에서 각각 입력**합니다

### 6.3 위치 정보 입력 우선순위

1. **위경도**: 논문에 좌표가 명시된 경우 반드시 입력
2. **행정구역**: 좌표가 없더라도 산출지 정보를 텍스트로 입력
3. 둘 다 없으면 비고에 가능한 정보 기입 (예: "태백 부근")

### 6.4 주의사항

- 논문 등록 전 **반드시 중복 확인** (제목, 저자, 연도로 검색)
- 저자명은 시스템에 등록된 형식과 일치해야 합니다 — 자동완성 활용
- PDF 업로드 시 저작권에 유의하세요 (내부 연구 목적)
- 입력 완료 후 **논문 상세 페이지**에서 전체 내용을 검토해 주세요

## 7. FAQ

### Q: 같은 화석산지가 여러 논문에 나옵니다. 어떻게 하나요?
A: 각 논문에서 독립적으로 표본/위경도를 입력하면 됩니다. 관리자가 위경도 기반으로 화석산지를 통합 관리합니다.

### Q: 논문에 좌표가 없고 지명만 있습니다.
A: 위치(행정구역) 필드에 지명을 입력하세요. 좌표는 비워둡니다.

### Q: 저자가 자동완성에 나오지 않습니다.
A: KPRDB > 저자목록에서 새 저자를 먼저 등록한 뒤, 논문 편집에서 다시 선택하세요.

### Q: 분류군(Taxon)과 학명의 차이는?
A: 학명(ScientificName)은 종 수준의 정식 명칭이고, 분류군(ReferenceTaxon)은 특정 논문에서 다루는 학명+지층+시대의 조합입니다. 같은 학명이 여러 논문에 등장할 수 있습니다.

### Q: 입력한 내용을 수정하고 싶습니다.
A: 논문 상세 페이지에서 [Edit] 버튼으로 수정 가능합니다. 모든 편집 이력은 시스템에 기록됩니다.

### Q: 논문의 PDF가 자동 처리되었다는데, 결과를 어디서 보나요?
A: 논문 상세 페이지 하단의 "PDF 추출 데이터" 섹션에서 자동 추출된 좌표와 표본번호를 확인할 수 있습니다.
