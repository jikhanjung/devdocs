# 국내 주요 화석산지 발굴 및 활용 중장기계획 수립 — 개발 관련 요약

**발주**: 국가유산청 | **수행**: 한국고생물학회 | **제안일**: 2026-02-22 | **용역기간**: 2026.03~2026.11 (8개월)

---

## 1. 사업 개요 및 목표

현재 화석산지 정보는 학술논문·지자체 보고서·소장기관 자료로 분산되어 있어 좌표 기반의 일관된 관리와 위험 대응이 어렵다. 해안절벽·노두·하천변 등 개발에 취약한 산지는 훼손·멸실 시 복원이 사실상 불가능하므로, **사전 진단과 우선순위 설정에 기반한 선제적 관리체계**가 필요하다.

본 용역의 4대 목표:

1. **화석산지 DB 구축** — 위경도 좌표 기반, 고유 산지 ID(GeoSite-ID) 부여, 데이터 품질 고도화
2. **화석표본 DB 구축** — 모식표본 중심, 분류·층서·보존상태·소장기관 정리, 북한·해외 포함
3. **GIS 통합 플랫폼 구현** — GeoHeritage Evaluator(가칭), 위성영상 기반 위험도(Risk Score) 산정, P0~P4 우선순위 도출
4. **중장기 로드맵 및 예산(안) 제시** — 연차별 조사–발굴–보존–연구–활용 계획

## 2. 핵심 데이터 구조

### ID 체계 (세 DB의 연동 축)

| ID | 대상 | 설명 |
|----|------|------|
| **GeoSite-ID** | 화석산지 | 고유 산지 식별자, 좌표·층서·보호현황 연동 |
| **Specimen-ID** | 화석표본 | GeoSite-ID와 연동, 소장기관·등록번호 포함 |
| **Reference-ID** | 문헌 | 표본·산지와 연결되는 근거 문헌 (DOI 포함) |

### 화석산지 DB 필드 (GeoSite-ID 단위)

- 위경도 (decimal degrees), 좌표 정확도 등급 A/B/C
  - A: 노두/지점 수준
  - B: 리/면 단위
  - C: 비공개 대체좌표
- 층서 정보 (지층명, 지질시대)
- 산출양상, 화석 종류
- 보호·지정 현황 (천연기념물 등)
- 위협요인
- 좌표 공개 수준: 공개 / 제한 / 비공개 (불법채집 방지)

### 화석표본 DB 필드 (Specimen-ID 단위)

- 분류학적 결론 (최신 학명, 동의어)
- 산출지 (GeoSite-ID 연동)
- 층서적 위치 (지층/연대)
- 보존양상
- 소장기관 및 등록번호
- 이미지/도면
- 근거문헌 (Reference-ID, DOI)
- 북한·해외 소재 표본은 분리 구축 (문헌/카탈로그 기반)

## 3. GeoHeritage Evaluator 플랫폼

GIS 기반 통합 조회 + 모니터링/위험평가 플랫폼 (웹 구현 예정)

### 주요 기능

- 화석산지 DB 지도 레이어 조회·검색·필터링
- 위성영상 시계열 비교 (ESRI Wayback 활용): 공사 흔적·범람원 확대 등 지표 변화 기록
- 토지피복 변화 및 개발 흔적 모니터링
- 모니터링 결과 → 위험도(Risk Score) 자동 입력 연계
- 발굴 우선순위(P0~P4) 시각화 및 대시보드
- 리포트 출력 기능
- 대상: DB 구축 화석산지 중 최대 200곳

## 4. 위험도 평가 기준 (Brilha, 2016)

정량적 위험도(Risk Score)는 5개 항목, 각 1~4점, 가중합으로 산정. **점수가 높을수록 위험 크다.**

| 평가 항목 | 가중치 | 점수 기준 |
|----------|--------|----------|
| **지질요소의 훼손(열화) 가능성** | **35%** | 4점: 전체 훼손 가능성 높음<br>3점: 주요 요소 훼손 가능<br>2점: 부차적 요소 훼손 가능<br>1점: 부차적 요소도 훼손 가능성 낮음 |
| **훼손 유발 활동·지역과의 근접성** | **20%** | 4점: 50m 미만<br>3점: 200m 미만<br>2점: 500m 미만<br>1점: 1km 미만 |
| **법적 보호 및 접근 통제** | **20%** | 4점: 보호 없음 + 접근통제 없음<br>3점: 보호 없음 + 접근통제 있음<br>2점: 보호 있음 + 접근통제 없음<br>1점: 보호 있음 + 접근통제 있음 |
| **접근성 (도로 접근 여건)** | **15%** | 4점: 포장도로 100m 미만 + 버스 주차 가능<br>3점: 포장도로 500m 미만<br>2점: 비포장도로로 버스 접근 가능<br>1점: 버스 가능 도로에서 1km 미만 |
| **인구 밀도 (주변 인구 압력)** | **10%** | 4점: 1,000명/km² 초과<br>3점: 250~1,000명/km²<br>2점: 100~250명/km²<br>1점: 100명/km² 미만 |

> **참고**: gheval 현재 구현은 접근성(도로거리)과 식생피복만 반영. 법적보호·인구밀도·훼손가능성 항목은 추가 구현 필요.

## 5. 발굴 우선순위 (P0~P4)

우선순위는 단일 기준이 아닌 **Risk × Value × Feasibility** 3축 결합으로 결정 (Sevieri et al., 2020).

| 축 | 출처 | 설명 |
|----|------|------|
| **Risk** (훼손·멸실 위험도) | GeoHeritage Evaluator 산출값 | Brilha 2016 기준 |
| **Value** (학술·문화 가치) | 표본 DB 중요성 평가 | 높음/중간/낮음 3등급 |
| **Feasibility** (실행가능성) | 현장 평가 | 인허가·접근·안전·보존처리 여건 |

### 우선순위 매트릭스 (Risk × Value)

|  | 가치 낮음 | 가치 중간 | 가치 높음 |
|--|----------|----------|----------|
| **위험도 높음** | **P2** 방어/기록 우선, 긴급 훼손저감, 발굴은 제한적 | **P1** 긴급조치+선별발굴, 1~2년 내 착수 | **P0** 최우선 긴급발굴/보존, 즉시 착수 |
| **위험도 중간** | **P3** 모니터링 중심 | **P2** 단계적 조사, 정밀조사 후 발굴 검토 | **P1** 계획발굴 우선, 2~3년 내 배치 |
| **위험도 낮음** | **P4** 관찰·기록, 예산 최소 | **P3** 기초연구/활용연계 | **P2** 장기 보전·활용, 관광/교육 적합 |

### 우선순위 정의

| 등급 | 정의 | 조치 |
|------|------|------|
| **P0** | 최우선·긴급 대응형 | 즉시 정밀조사·긴급발굴(또는 긴급 보존조치) 착수 |
| **P1** | 우선 추진형 | 단기(1~2년) 내 발굴·연구 우선 배치, 보존·관리 계획 병행 |
| **P2** | 단계 추진형 | 중기(2~3년) 내 정밀조사 후 필요 시 발굴 전환 |
| **P3** | 모니터링·기초조사형 | 정기 모니터링·기초조사 중심, 상황 변화 시 상향 조정 |
| **P4** | 관찰·기록 유지형 | DB 기록 유지와 최소 관찰, 예산 최소 투입 |

## 6. 과업 수행계획 (5단계)

### 1단계: 착수 및 요구사항 확정 (착수~1개월)
- 대상 화석산지 범위 및 민감 좌표 처리 원칙 확정
- DB 표준 확정: 메타데이터 규격, 필수/선택 필드, 좌표체계, 품질등급, 버전관리
- GeoHeritage Evaluator 기능 범위 확정: 지도 레이어, 필터·검색, 대시보드, 변화기록, 리포트 출력
- **산출물**: 착수보고서, 세부수행계획, **데이터 표준서**

### 2단계: DB 구축을 위한 데이터 수집 (1~4개월)
- 국내 고생물학 논문(526편) 기반 표본·산지 DB 1차 구축
- Specimen-ID 부여: 분류·층서·보존·소장·이미지·문헌 통합
- GeoSite-ID 부여 + 좌표 표준화(decimal degrees) + 정확도 등급(A/B/C)
- 북한·해외 소재 표본은 분리 구축 (문헌/카탈로그 기반)
- **산출물**: 표본 DB 1차본, 화석산지 DB 1차본, **산지 분포 GIS 레이어**

### 3단계: 표본 중요성 평가 기능 탑재 (3~4개월)
- "발굴된 매장문화재 평가표" 기반 모식표본 중요성 평가 항목·채점 가이드 마련
- 플랫폼 내 중요성 평가 입력 기능 구현
- 가치 등급: 높음 / 중간 / 낮음 3단계
- **산출물**: 중요성 평가 운영지침, 평가결과(시범군)

### 4단계: GeoHeritage Evaluator 구축 및 위험도/우선순위 도출 (4~7개월)
- 위성영상 기반 토지피복 변화·개발 흔적 기록 (정성 평가)
- 접근성(도로망)·인구밀도 반영 정량 위험도 평가
- Value × Risk 결합 → P0~P4 우선순위 도출
- **산출물**: GeoHeritage Evaluator 베타버전, **발굴 우선순위 평가표**, 연차별 발굴·연구계획(초안)

### 5단계: 관리·활용 방안 및 최종 납품 (7~8개월)
- 연구 완료 표본 보존·관리·전시·활용 방안(프로토콜/권고안)
- **전국 화석지도(디지털/인쇄)** 제작
- 중장기계획 단위 분해 + 예산(안) 산정
- **산출물**: 최종보고서, DB(산지/표본), GeoHeritage Evaluator, 운영매뉴얼, 예산(안), 전국 화석지도

## 8. 기대효과 및 활용방안

### 기대효과

**정책·행정**: P0~P4 체계로 사후 대응형 → 계획적 관리체계 전환, 훼손 가능성 조기 탐지

**보존·관리**: 산지별 변화 누적 기록, 모식표본 관리체계 표준화 (보존처리·이관·열람·전시 효율화)

**학술·연구**: 분산된 산지/표본 정보 통합 → 연구 공백 분석, 모식표본 추적성 강화로 재현성 개선

**사회·문화**: 전국 화석지도 제작, 지역 박물관·지질공원·교육기관 연계. **민감좌표는 공개 수준 차등화** (훼손 위험 방지)

### 활용방안

| 활용 주체 | 주요 활용 내용 |
|----------|--------------|
| 국가유산청 | 우선순위 기반 투자 결정, 연차계획·예산 편성, KPI 성과관리 |
| 지자체·현장 관리주체 | 접근·안전·출입 관리, 개발협의 시 객관적 근거 자료 |
| 소장기관·연구자 | 모식표본 관리·이관·열람·전시 계획 지원, 국외 협력 과제 도출 |
| 대국민 | 전국 화석지도 + 해설 콘텐츠, 교육·체험·전시, 온라인 디지털 컬렉션 |
