# 044. 북한 논문 PDF 일괄 임포트

**날짜**: 2026-03-27
**버전**: 0.2.82

## 개요

`nkpaleo/` 디렉토리에 업로드된 북한 고생물학 논문 PDF 96건을 DB와 매칭하여 연결.

## 업로드 파일

- `nkpaleo/`: 논문 PDF 92건
- `nkpaleo/Book/`: 단행본 PDF 4건
  - 조선의화석1 고생대편 (1987, 55MB)
  - 조선의화석2 중생대편 (1987, 89MB)
  - 조선고생물화석 (1997, 253MB)
  - Sinuiju Biota of the Mesozoic in Korea (2022, 14MB)

## 매칭 작업

### 1단계: 기존 DB 엔트리와 매칭 (61건 → 55건 PDF 연결)

- 한글 파일명에서 저자, 연도, 제목 파싱
- DB의 영문 제목/remarks와 키워드 매칭 (지명, 화석종류, 지질시대)
- 1:1 확실한 매칭 55건 → PDF 연결 + `title_k` 한글 제목 업데이트
- 1:N 매칭 문제: 같은 저자+연도에 복수 논문 → 제목 키워드로 구분

### 1:N 매칭 해결 사례

- `박용선 1966`: (1)=완족류 화석에 대하여 → Pak (1966a), (2)=지질 시대 → Pak (1966b)
- `김덕성 1984`: (1) → Kim (1984a), (2) → Kim (1984b)
- `박용선+강진건 1984`: 데본기 식물 → Pak and Kang (1984a), 해백합 → Pak and Kang (1984b)

### 2단계: 신규 논문 생성 (32건)

DB에 없던 논문 32건 신규 생성 + PDF 연결:
- 리죽남 (1961, 1963×2, 1993, 2000)
- 박명화 (1993, 2006, 2016)
- 박용선 (1986, 1987×2, 1992, 1966 림진계)
- 리성범+전민우 (1971), 최문경 (1971), 전민우+진영필 (1972)
- 김영길 (1988), 리석률 (1988), 리봉상 (1995)
- 강진건 (2019), 강준철+최일영 (2012), 강진건+김동철 (2012)
- 김명학+김명국 (2007, 2008), 김명학 (2017)
- 김병성+한성룡 (2014), 김성호+리철준 (2016)
- 리은빛 (2018), 리철준 (2021)
- 서광석+정명훈+원철국 (2017), 김덕성+강진건 (1993)
- 김덕성 (연도 불명)

### 3단계: 추가 누락 5건

1:N에서 빠진 것이 아니라 DB에 없던 신규:
- 강준철+서광식+원철국 (2015) 흔적화석
- 원철국 (2020) 곤충화석
- 원철국+서광식 (2015) 약충화석 Aeschnidiid
- 원철국+서광식 (2016) 조개류화석
- 원철국+서광식 (2020) 백토동 고생물

### 4단계: Book 디렉토리 (4건)

- 조선의화석1: 기존 ID=3536에 PDF 연결 + 한글 제목
- 조선의화석2: 신규 생성
- 조선고생물화석 (1997): 신규 생성
- Sinuiju Biota (2022): 신규 생성
